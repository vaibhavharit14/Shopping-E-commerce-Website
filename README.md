# 💎 Forever Luxury E-commerce Platform

A state-of-the-art, AI-enhanced e-commerce solution built with the MERN stack. Designed with a premium aesthetic, featuring glassmorphism, dynamic animations, and a seamless user experience.

![Preview](https://via.placeholder.com/1200x600?text=Forever+Luxury+E-commerce+Showcase)

## 🚀 Elevated Features

- **Premium UI/UX**: Crafted with React & Framer Motion for buttery-smooth transitions and a "high-end" feel.
- **Smart Product Management**: Supports both local asset synchronization and dynamic backend API products.
- **Advanced Filtering**: Precision search and category/subcategory filtering for effortless discovery.
- **Secure Checkout**: Integrated with Razorpay/Stripe (Demo Mode) and Cash on Delivery.
- **Admin Command Center**: Complete control over products, orders, and promotion/coupon management.
- **Smart Coupons**: Dynamic promo code system with expiry tracking and global application.
- **Real-time Order Tracking**: Visual progress timeline for customer transparency.

## 🛠️ Technology Stack

| Layer | Technologies |
|---|---|
| **Frontend** | React 19, Framer Motion, TailwindCSS, Lucide Icons, Axios |
| **Backend** | Node.js, Express, JWT, BcryptJS, Multer (File Uploads) |
| **Database** | MongoDB with Mongoose ODM |
| **Security** | Helmet.js, Express Rate Limit, JWT-based Authentication |

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-repo/ai-ecommerce-project.git
   cd ai-ecommerce-project
   ```

2. **Environment Configuration**
   - Configure `.env` files in both `frontend` and `backend` directories (see `.env.example`).

3. **Install Dependencies**
   ```bash
   # From the root directory
   npm install && cd backend && npm install && cd ../frontend && npm install
   ```

4. **Launch Platform**
   ```bash
   # Start both Server & Client concurrently
   npm run dev
   ```
